### ButtonGroup Component

The `ButtonGroup` component is used for group buttons. It can be customized using various props.

```js
  orientation?: "row" | "column";
  spacing?: "none" | "tight" | "normal" | "loose" | "extra-loose";
}
```

#### Example Usage `ButtonGroup`

```tsx
<ButtonGroup orientation="row" spacing="tight">
    <Button label="Button 1" className="r-md" />
    <Button label="Button 2" />
    <Button label="Button 3" className="r-md" />
</ButtonGroup>
```
