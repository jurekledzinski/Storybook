### IconButton Component

The `IconButton` component is used for triggering actions. It can be customized using various props.

#### Props (typescript)

Hereâ€™s a list of the base props for the `IconButton` or `LinkIconButton` component:

```js
{
  border?: 'border-xs' | 'border-sm' | 'border-md' | 'border-lg' | 'border-xl';
  icon: React.ElementType | IconDefinition;
  isLoading?: boolean;
  disabled?: boolean;
  fullWidth?: boolean;
  radius?: 'radius-xs' | 'radius-sm' | 'radius-md' | 'radius-lg' | 'radius-xl';
  size?: 'size-xs' | 'size-sm' | 'size-md' | 'size-lg';
  color: 'primary' | 'secondary' | 'success' | 'negative' | 'warning';
  variant: 'contained' | 'outlined' | 'text';
}
```

Prop necessery to switch to `LinkIconButton`:

```js
{
    href?: string;
}
```

#### Default set props

```js
{
  border: 'border-xs',
  color: 'primary',
  radius: 'radius-none',
  size: 'medium',
  variant:'contained',
}
```

#### Example Usage `IconButton`

```tsx
<IconButton label="Click me" />
```

#### Example Usage `IconLinkButton`

```tsx
<IconButton label="Click me" href="/" />
```
