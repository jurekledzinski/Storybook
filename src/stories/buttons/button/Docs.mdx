### Button Component

The `Button` component is used for triggering actions or submitting forms. It can be customized using various props.

#### Props (typescript)

Hereâ€™s a list of the base props for the `Button` or `LinkButton` component:

By `className` can add:

- border: 'b-xs' | 'b-sm' | 'b-md' | 'b-lg';
- radius: 'r-xs' | 'r-sm' | 'r-md' | 'r-lg' | 'r-xl';

```js
{
  iconStart?: React.ElementType | IconDefinition;
  iconEnd?: React.ElementType | IconDefinition;
  isLoading?: boolean;
  disabled?: boolean;
  fullWidth?: boolean;
  size?: 'size-xs' | 'size-sm' | 'size-md' | 'size-lg';
  color?: 'info' |'primary' | 'secondary' | 'success' | 'negative' | 'warning' | 'white';
  variant?: 'contained' | 'outlined' | 'text';
  label: string;
}
```

Prop necessery to switch to `LinkButton`:

```js
{
    href?: string;
}
```

#### Default set props

```js
{
  color: 'primary',
  size: 'medium',
  variant:'contained',
}
```

#### Example Usage `Button`

```tsx
<Button label="Click me" />
```

#### Example Usage `LinkButton`

```tsx
<Button label="Click me" href="/" />
```
