### Button Component

The `Button` component is used for triggering actions or submitting forms. It can be customized using various props.

#### Props (typescript)

Hereâ€™s a list of the base props for the `Button` or `LinkButton` component:

```js
{
  border?: 'border-xs' | 'border-sm' | 'border-md' | 'border-lg' | 'border-xl';
  iconStart?: IconElement<Omit<IconProps, 'icon'>>;
  iconEnd?: IconElement<Omit<IconProps, 'icon'>>;
  isLoading?: boolean;
  disabled?: boolean;
  fullWidth?: boolean;
  radius?:
    | 'radius-none'
    | 'radius-xs'
    | 'radius-sm'
    | 'radius-md'
    | 'radius-lg'
    | 'radius-xl'
    | 'radius-x2'
    | 'radius-x3'
    | 'radius-full';
  size?: 'small' | 'medium' | 'large';
  color?: 'primary' | 'secondary' | 'success' | 'negative' | 'warning';
  type?: 'button' | 'reset' | 'submit';
  variant?: 'contained' | 'outlined' | 'text';
  label: string;
}
```

Prop necessery to switch to `LinkButton`:

```js
{
    href?: string;
}
```

#### Default set props

```js
{
  border: 'border-xs',
  color: 'primary',
  radius: 'radius-none',
  size: 'medium',
  variant:'contained',
}
```

#### Example Usage `Button`

```tsx
<Button label="Click me" />
```

#### Example Usage `LinkButton`

```tsx
<Button label="Click me" href="/" />
```
