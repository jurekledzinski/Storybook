### Alert Component

The `Alert` component is used to display brief, important messages to the user.
It provides contextual feedback such as success, error, warning, or informational messages.

Use alerts to communicate status updates, confirmations, or critical issues that require the user’s attention.

#### Props (typescript)

Here’s a list of the props for the `Alert` component:

By `className` can add:

- radius: 'r-xs' | 'r-sm' | 'r-md' | 'r-lg' | 'r-xl';

```js
{
  children?: React.ReactNode;
  className?: string;
  color?: "info" | "primary" | "secondary" | "success" | "negative" | "warning" | "white";
  fullWidth?: boolean;
  orientation?: 'row' | 'column';
  variant?: 'contained' | 'filled' | 'light' | 'outlined';
}
```

#### Example Usage `Alert`

```tsx
<Alert color="info" variant="contained">
  <AlertIcon icon={faCircleInfo} />
  <AlertMessage message="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
  <AlertButton color="white" variant="text" />
</Alert>
```
