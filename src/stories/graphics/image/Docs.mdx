### Image Component

The `Image` component renders an optimized and responsive image element.  
It supports lazy loading, fallback behavior, and customizable styling.

#### Example usage `Image`

```tsx
<Image src="/assets/example.jpg" alt="Example Image" width={400} height={300} loading="lazy" />
```
