### FilePreviewList Component

`FilePreviewList` is a component that displays a list of file previews (such as images). It accepts an array of files or array of data and renders each file using a customizable renderItem function. If no files are available, it displays the fallback content provided as children.

```js
  placement?: 'column' | 'grid' | 'row';
}
```

```tsx
<PreviewList images={[]}>
    {(file, index, placement) => (

<PreviewList/>
```
