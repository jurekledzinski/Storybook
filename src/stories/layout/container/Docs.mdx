### Container Component

The `Container` component is a layout wrapper used to center and constrain the width of content within a page or section.
It provides consistent horizontal padding and maximum width, ensuring your UI looks clean and aligned across different screen sizes.

Typically used as a top-level layout element in pages, forms, or sections where content should not stretch to the full viewport width.
