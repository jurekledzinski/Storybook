### Form Component

The `Form` component serves as a wrapper for `FormGroup` and `Field` components, providing structure and submission handling for form elements. It ensures consistent layout across all form groups.

Hereâ€™s a list of the props for the `Form` component:

```js
  orientation: 'row' | 'column';
  ...FormHTMLAttributes<HTMLFormElement>
}
```
#### Example Usage `Form`

```tsx
<Form className="g-sm">
    <FormGroup>
        <Field>
          <Label>Name</Label>
          <Input />
          <Message color="negative">Name is required</Message>
        </Field>
    </FormGroup>
    <FormGroup>
        <Field>
          <Label>Surname</Label>
          <Input />
          <Message color="negative">Surname is required</Message>
        </Field>
    </FormGroup>
</Form>
```
