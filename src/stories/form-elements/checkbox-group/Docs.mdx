### CheckboxGroup Component

`CheckboxGroup` is used when you want users to select multiple options from a list.

#### Example Usage `CheckboxGroup`

```tsx
<CheckboxGroup spacing="normal">
  <Checkbox id="apple" name="fruits" value="apple">
    Apple
  </Checkbox>
  <Checkbox id="mango" name="fruits" value="mango">
    Mango
  </Checkbox>
  <Checkbox id="kiwi" name="fruits" value="kiwi">
    Kiwi
  </Checkbox>
  <Checkbox id="banan" name="fruits" value="banan">
    Banan
  </Checkbox>
</CheckboxGroup>
```
