# Border Utilities

These CSS utility classes define the standard border, radius system used across components and layouts.

They are based on CSS variables for consistent border, radius across the application.

## Border variables css

- --border-xs: 1px;
- --border-sm: 2px;
- --border-md: 3px;
- --border-lg: 4px;

## Radius variables css

- --radius-xs: 2px;
- --radius-sm: 4px;
- --radius-md: 6px;
- --radius-lg: 8px;
- --radius-xl: 100vh;

## Border (b) classNames

- b-xs: `1px`;
- b-sm: `2px`;
- b-md: `3px`;
- b-lg: `4px`;

## Radius (r) classNames

- r-xs: `2px`;
- r-sm: `4px`;
- r-md: `6px`;
- r-lg: `8px`;
- r-xl: `100vh`;
