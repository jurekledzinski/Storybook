# Spacing Utilities

These CSS utility classes define the standard spacing system used across components and layouts.

They are based on CSS variables for consistent spacing and sizing across the application.

---

## Spacing variables css

- --space-initial: `initial`;
- --space-xxs: `2px`;
- --space-xs: `4px`;
- --space-sm: `8px`;
- --space-md: `16px`;
- --space-lg: `24px`;

## Max width variables css

- --max-w-sm: `1000px`;
- --max-w-md: `1100px`;
- --max-w-lg: `1200px`;
- --max-w-xl: `1300px`;
